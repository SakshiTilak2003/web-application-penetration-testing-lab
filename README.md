# Web Application Penetration Testing Lab

This project demonstrates a hands-on penetration testing assessment conducted on OWASP Juice Shop in a controlled lab environment.

## Lab Setup

Attacker Machine: Parrot OS  
Target Machine: Windows 11  
Virtualization: VirtualBox  
Network: Internal Network

## Tools Used

Nmap  
Burp Suite  
Nikto  
SQLMap

## Vulnerabilities Identified

- SQL Injection
- Reflected XSS
- DOM-Based XSS
- Broken Access Control

## Project Outcome

The project resulted in a professional penetration testing report including vulnerability findings, risk classification, and remediation strategies.
